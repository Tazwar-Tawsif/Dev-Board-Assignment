Question-1: What are the different ways to select an element in the DOM?
There are several ways to select an element.
1. getElementByID() which selects an element with an 'id'
2. getElementByClassName() which selects an element with a 'class'
3. getElementByTagName() which selects an element with its 'tag name'
4. querySelector() which is a CSS selector, can be used with 'id', 'class'
5. querySelectorAll() which selects all the elements with the same 'class' name or tag name

Question-3: What is event delegation in the DOM?
Simply it is a technique where you add one event listener to the parent and events in the children can be handled with that one event listener. Helpful in the case of adding/removing unknown new elements in a parent. Also helps with clean coding.
Question-4: What is event bubbling in the DOM?
It is a technique related with event delegation where an event starts from a target element in a child and bubbles up to the parent element to respond to that particular event.
Question-5: How do you create, add, and remove elements using JavaScript?
Lets say we want to add a paragraph in a div and remove a heading from that div.
##Create
const new_paragraph = document.createElement('p'); new_paragraph.innerText = "hello new paragraph";
##Add
document.getElementByTagName('div').appendChild(new_paragraph);
##Remove
document.getElementByTagName('div').removeChild(document.getElementByID('heading'));
